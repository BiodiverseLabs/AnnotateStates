# AnnotateStates
A Python script and shapefiles to annotate US states to an excel file from latitude and longitude.

Shapefiles were attained from the US Census Bureau: TIGER/Line Shapefiles.
https://www.census.gov/geographies/mapping-files/time-series/geo/tiger-line-file.html

Usage:
Unzip and drop the shapefiles, your target Excel sheet with lat/long, and python file into a folder. 
Update the file locations within the Python files.
Run the Python script.
